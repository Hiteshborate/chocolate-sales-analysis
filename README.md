# Chocolate Sales Analysis

Data analysis project on chocolate sales from December 2023 to February 2024.

## Dataset

- 443 transactions
- 26 products
- 4 purchase channels (Website, App, In Store, Phone)
- 14 customer segments (job titles)

## What's Inside

**1. EDA (Excel)**
Exploratory data analysis with stats, monthly breakdown, product performance, and channel analysis.

**2. Dashboard (HTML)**
Interactive Power BI-style dashboard with charts and KPIs. Open in any browser.

**3. Summary Report (PDF)**
5-page executive report with key findings and recommendations.

## Key Numbers

- Total Revenue: Rs. 2,13,550
- Conversion Rate: 73.6%
- Top Product: Organic Choco Syrup
- Best Channel: Website (Rs. 81,070)
- Peak Month: December 2023 (Rs. 83,605)
